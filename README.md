## 📜 프로젝트 개요

**여기로 모여라** 프로젝트는 사용자가 실시간으로 위치를 공유하여 약속 장소에 누가 도착했는지 확인할 수 있는 앱입니다.

이 프로젝트는 **앱 개발자 2명**과 **서버 개발자 1명**으로 구성된 팀이 진행하고 있습니다. 

## 🛠 기술 스택

### Back-End
- **Java**
- **Spring Boot**
- **JPA**
- **MySQL**
- **EC2**
- **Docker**
- **Redis**
- **WebSocket**
- **Spring Security**
